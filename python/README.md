## Specific Instructions for Python

*   Modify the top few lines of `main.py` only as necessary to import the
    modules you've written.
*   Run the program using `python main.py`.

### Necessary Declarations

Following are all of the publically-accesible methods and functions you'll need
to expose through import statements in the `main.py` file:

*   `CircularRobot.__init__()` (constructor)
*   `RectangularRobot.__init__()` (constructor)
*   `is_colliding(a: Robot, b: Robot) -> bool`
